Evolving to CSS3

New features of CSS3

In this article we will discuss the exciting new features of CSS3. I will cover everything from the new animations to the added selectors.

Here are some the new features that are included in CSS3.


Selectors

Since selectors are at the core of CSS, they have been expanded to an extent. The attribute selectors, introduced in CSS2, have a new mechanism where we don't have to type the full value of the attribute. For example, we can use a[href^="http://sales"] to find links whose sources begin with sales; we can also use $= and *= to signify ends with and contains respectively. Another type of selectors that got extended are pseudo-selectors (:hover, :visited). For example, :last-child has been added as a compliment to :first-child. Visit DevDocs for more.


Colors

CSS3 offers up new methods of picking colors like hsl, hsla and rgba which have the ability to declare semi-transparent colors. rgba works like rgb with an added parameter for alpha (opacity). hsl stands for hue, saturation and lightness allowing for advanced manipulation while hsla just adds opacity to HSL. Visit DevDocs for more.


Animations and Text Effects

Animations that do not require JavaScript have been included in CSS3. To get it working, you must first define a keyframe which acts like a function. Take the following as an example.
@keyframes example {
  from {background-color: red;}
  to {background-color: yellow;}
}
Then you should bind it to an element to get it working using the syntax animation: example; . You can even set the duration with the animation-duration property. To make your animation based on user input you can couple it with the pseudo-selector :hover .

Another new feature is the text effects. These include attributes text-overflow, word-wrap, word-break, writing-mode . Visit W3C Schools for more.


Viewport Height / Width

In CSS3 you can use new relative sizes vw and vh, viewport width and viewport height respectively. 1 vw / vh equals 1 / 100 times the viewport's width / height. Visit DevDocs for more.


Rounded Corners

CSS3 features a new attribute called border-radius that enables you to curve your border. At 100%, it will turn into a circle. Visit DevDocs for more.


Gradients

Using this new syntax background-image: linear-gradient(direction, color-stop1, color-stop2, ...); you will be able to apply linear gradients as a background image. For radial gradients use the syntax background-image: radial-gradient(shape size at position, start-color, ..., last-color); . Visit W3C Schools for more.


Multiple Background Images

CSS3 now supports multiple background images. To use this feature just follow this syntax: background-image: url(...), url(...), ...; . Visit W3C Schools for more.
